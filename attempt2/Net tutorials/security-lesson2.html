Lesson 2: Introduction to Virtual Machines

What is a Virtual Machine?

A virtual machine is a virtual computer that can be installed into a physical computer. It emulates a complete computer system, allowing you to run different operating systems on the same hardware.

Benefits of Virtual Machines:

  • Cost-effective: Run multiple computers on one physical machine
  • Safe learning environment: Experiment without risking your main system
  • Easy recovery: Delete and recreate VMs if infected
  • Compatibility: Run older or different operating systems
  • Isolation: Test malware or suspicious software safely

How to Install a Virtual Machine (VirtualBox)

This guide is for Windows 10 systems installing VirtualBox with Kali Linux.

Required Resources:

  • VirtualBox software
  • Kali Linux ISO image
  • 7-Zip (for file extraction)

Step 1: Download VirtualBox

Visit: https://www.virtualbox.org/wiki/Downloads

Click "Windows hosts" to download VirtualBox 7.0.18 or newer

Step 2: Download Kali Linux

Visit: https://www.kali.org/downloads/

Click "Virtual Machines" → Select "VirtualBox 64-bit"

File will be named: kali-linux-2024.2-virtualbox-amd64.ova

Step 3: Install 7-Zip (if needed)

Visit: https://www.7-zip.org/

Download the 64-bit version for your system

Step 4: Set Up VirtualBox

  1. Run the VirtualBox installer and complete setup
  2. Open VirtualBox and click "Import"
  3. Select the downloaded Kali Linux file (.ova)
  4. Follow the import wizard (use default settings)
  5. Start your new virtual machine

Step 5: Log In to Kali Linux

Default credentials:

Username: kali

Password: kali

Security Tips for Virtual Machines:

  • Take snapshots before making major changes
  • Isolate network connections when testing malware
  • Keep your VM software updated
  • Don't use the same passwords as your host machine
  • Disable shared folders/clipboard when not needed
